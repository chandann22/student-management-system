# Student Management System

A database-driven application developed using Python and SQL for efficient student record management.

## Features

* Student Record Management
* CRUD Operations
* Login Authentication
* Attendance Tracking
* Grade Management
* SQL Database Integration

## Technologies Used

* Python
* SQL

## Project Overview

The system allows administrators to securely manage student information, track attendance, and maintain academic records using a database backend.
